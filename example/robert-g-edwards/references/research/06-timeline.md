# 06 - 人物时间线

| 时间 | 事件 | 影响 |
|------|------|------|
| 1925.09.27 | 出生于Yorkshire Batley，工人阶级家庭 | 阶级意识、平等主义、约克郡性格 |
| ~1937 | 进入Manchester Central Boys' High School | 教育改变命运 |
| 1943-1948 | 二战期间服役于英国陆军（北非、意大利） | 战争体验塑造韧性 |
| 1948 | 进入University College of North Wales, Bangor | 初始学农业，后转动物学 |
| 1951 | 获得B.Sc.学位 | 开始研究生涯 |
| 1951-1952 | CalTech工作一年 | 接触美国前沿科学 |
| 1955 | Edinburgh大学获Ph.D. | 动物遗传学基础 |
| 1962 | 获D.Sc.学位 | 学术地位确立 |
| 1963 | 加入Cambridge University | 终身学术基地 |
| 1965-1968 | 人类卵子体外受精初步成功 | 科学可行性证明 |
| 1966 | 与Pincus公开辩论（Venice） | 不畏权威 |
| 1968 | 遇见Patrick Steptoe，开始合作 | IVF项目的转折点 |
| 1969 | 发表首篇人类卵子体外受精论文 | 学术里程碑 |
| 1971 | MRC拒绝资助 | 转向私人资助，承受压力 |
| 1970s | 遭受大量专业批评和媒体争议 | 主动发起公众伦理教育 |
| 1978.07.25 | Louise Brown出生 — 世界首个IVF婴儿 | 证明成功，改变医学史 |
| 1980 | 与Steptoe合著《A Matter of Life》 | 公开记录全过程 |
| 1980 | 创立Bourn Hall Clinic | IVF技术临床化 |
| 1985 | 创立ESHRE，任首任主席 | 建立学术共同体 |
| 1986 | 创办Human Reproduction期刊 | 建立学术平台 |
| 1988 | Patrick Steptoe去世 | 失去最关键的合作伙伴 |
| 2000 | 创办Reproductive BioMedicine Online | 持续推动领域 |
| 2010.10.04 | 获诺贝尔生理学或医学奖 | 迟来的认可（32年后）|
| 2013.04.10 | 去世，享年87岁 | — |

## 思想转折点
1. **Bangor转动物学**: 从农业到生殖科学的转向
2. **遇见Steptoe (1968)**: 从纯科学转向临床应用
3. **MRC拒绝 (1971)**: 从体制内研究者变成体制外maverick
4. **Louise Brown (1978)**: 从争议人物变成被认可的开创者
