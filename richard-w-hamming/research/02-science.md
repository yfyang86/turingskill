# Richard W. Hamming - Scientific Contributions

## Error-Correcting Codes

### Hamming Codes (1950)
- Perfect single-error-correcting codes
- Can detect and correct single-bit errors
- Hamming(7,4) code: 4 data bits, 3 parity bits
- Foundation of modern coding theory
- Essential for reliable digital communication

### Hamming Distance
- Metric measuring difference between strings
- Number of positions at which symbols differ
- Fundamental to coding theory and information theory
- Used in DNA sequence analysis
- Machine learning similarity measures

### Applications
- Computer memory (ECC RAM)
- Satellite communications
- Deep space communication (Voyager, etc.)
- Data storage systems
- Network protocols

## Numerical Analysis

### Computational Approach

- Practical computation on early computers
- Bridged pure and applied mathematics

## Digital Signal Processing

### Hamming Window
- Smoothing function for signal processing
- Reduces spectral leakage in Fourier analysis
- Widely used in digital filter design
- Alternative to rectangular window

### Spectral Analysis
- Methods for frequency domain analysis
- Applications in communications
- Radar signal processing
- Audio and image processing
