# Donald E. Knuth - Ethical Views

## Open Source Before It Was Named

**TeX Public Domain Release**
- Released TeX source code freely before "open source" term existed
- Made complete source available to everyone
- Encouraged modifications and derivatives with naming requirement
- Early success story for what became the open-source movement

**Principles Behind Free Release**
- "I wanted to make it available to everybody"
- Believed software should serve the community
- Quality matters more than commercial success
- Long-term stability over feature expansion

**The Naming Condition**
- Modified versions must use different names (LaTeX, ConTeXt, etc.)
- Preserved integrity of original TeX
- Allowed vibrant ecosystem of derivatives
- Protected users from incompatible implementations

## The $2.56 Checks for Bugs

**Origin**
- One "hexadecimal dollar" ($2.56 = 2^8 cents)
- Rewards for errors found in TAOCP books
- Extended to TeX bug rewards

**Evolution**
- TeX bug rewards started at $2.56, doubled yearly
- Capped at $327.68 after seven doublings
- Account hacked in 2008s; switched to "Bank of San Serriffe" notes
- Checks became prized trophies, many framed rather than cashed

**Philosophy**
- Transparency and accountability
- Acknowledges human fallibility
- Turns error-finding into collaborative improvement
- "Beware of bugs in the above code; I have only proved it correct, not tried it."

## Patent-Free Stance

**Opposition to Software Patents**
- Spoken against patenting mathematical algorithms
- Believed ideas should be freely available
- Mathematics belongs to humanity
- "Patents are incompatible with the academic tradition"

**TeX Patent Policy**
- No patents on TeX technology
- Font design algorithms freely available
- Typography innovations in public domain

## Digital Preservation Concerns

**Archival Philosophy**
- TeX designed to be technology-independent
- Wanted documents readable for centuries
- "Archival in the sense that it was independent of changes in printing technology"
- Concerned about digital obsolescence

**Document Longevity**
- TeX source files remain readable decades later
- ASCII-based format ensures future accessibility
- Contrast with proprietary binary formats

## Version Number Philosophy

**π Convergence for TeX**
- Version 3 (1989)
- Version 3.1, 3.14, 3.141, 3.1415...
- Converging toward π
- Symbolizes perfection never fully reached

**e Convergence for Metafont**
- Similar approach with mathematical constant e
- Reflects philosophical commitment to asymptotic improvement
- Art metaphor: "A work of art is never finished, merely abandoned"

## Academic Integrity

**Attribution Standards**
- Careful citation and credit in all works
- Extensive historical research for TAOCP
- Acknowledges predecessors generously
- "Standing on the shoulders of giants"

**Reproducibility**
- All algorithms fully specified
- MIX/MMIX code allows verification
- Detailed analysis of correctness proofs
- Encourages independent verification

## Humility and Accessibility

**Replying to Everyone**
- Famous for answering all correspondence (pre-email)
- Now reachable through email channel on his website
- Takes time to respond thoughtfully
- Values communication with learners at all levels

**Teaching Philosophy**
- Knowledge should be shared freely
- Writing for future generations
- Clarity over cleverness
- "I don't have time to write shorter letters"
