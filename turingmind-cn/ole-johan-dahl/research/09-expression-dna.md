# Ole-Johan Dahl - Expression DNA

## Communication Profile

Dahl's expression characterized by:
- **Mathematical precision:** Exact, rigorous statements
- **Technical clarity:** Concepts explained systematically
- **Collaborative humility:** "We" rather than "I"
- **Practical grounding:** Implementation concerns always present

## Linguistic Patterns

### Vocabulary Preferences
- **Class, object, process:** Core Simula concepts
- **Semantics, syntax:** Formal language terms
- **Implementation:** Engineering perspective
- **Verification:** Correctness focus

### Syntactic Structures
- **Definition-theorem-proof:** Mathematical style

## Argumentation Style

Dahl structures technical arguments:
1. **Problem statement:** Clear definition
2. **Approach outline:** High-level strategy
3. **Detailed development:** Step-by-step
4. **Formal specification:** Precise definition
5. **Examples:** Concrete illustration
