# Yann LeCun - Key Writings and Publications

## Foundational Papers

### "Backpropagation Applied to Handwritten Zip Code Recognition" (1989)
- **Authors:** LeCun, Boser, Denker, Henderson, Howard, Hubbard, Jackel
- **Journal:** Neural Computation
- **Significance:** First practical CNN (LeNet-1)
- **Impact:** Validated CNNs for vision
- **Citations:** 15,000+

### "Optimal Brain Damage" (1990)
- **Co-authors:** John Denker, Sara Solla
- **Conference:** NIPS/NeurIPS
- **Significance:** Network pruning techniques
- **Impact:** Modern model compression

### "Gradient-Based Learning Applied to Document Recognition" (1998)
- **Co-authors:** Léon Bottou, Yoshua Bengio, Patrick Haffner
- **Journal:** Proceedings of the IEEE
- **Significance:** Definitive LeNet-5 paper
- **Citations:** 40,000+

### "Deep Learning" (2015)
- **Co-authors:** Yoshua Bengio, Geoffrey Hinton
- **Journal:** Nature
- **Significance:** Comprehensive survey
- **Authors:** Turing Award trio
- **Citations:** 30,000+

## Key Books

### "Deep Learning" (2016)
- **Co-authors:** Ian Goodfellow, Yoshua Bengio, Aaron Courville
- **Contribution:** Chapter on CNNs, computer vision
- **Publisher:** MIT Press
- **Availability:** Free online
- **Impact:** Standard reference

### "When Machines Learn" (French, 2019)
- **Original:** "Quand la machine apprend"

## Technical Papers

### "Convolutional Networks for Images, Speech, and Time Series" (1995)
- **Co-author:** Yoshua Bengio
- **Handbook:** The Handbook of Brain Theory
- **Content:** CNN tutorial
- **Impact:** Educational resource

### "The MNIST Database" (1998)
- **Co-creators:** Corinna Cortes, Christopher Burges
- **Content:** Handwritten digit dataset
- **Impact:** Standard benchmark
- **Usage:** Every ML course

### "A Tutorial on Energy-Based Learning" (2006)
- **Chapter:** Predicting Structured Data
- **Content:** EBMs for structured prediction
- **Impact:** Self-supervised learning theory

## ImageNet and Deep Learning

### "ImageNet Classification with Deep Convolutional Neural Networks" (2012)
- **Note:** Alex Krizhevsky, Ilya Sutskever, Geoffrey Hinton
- **LeCun's role:** CNN pioneer whose work enabled this
- **Impact:** Deep learning revolution trigger

### "OverFeat: Integrated Recognition, Localization and Detection" (2014)
- **Co-authors:** Sermanet, Eigen, Zhang, Mathieu, Fergus
- **Conference:** ICLR
- **Significance:** Unified vision tasks
