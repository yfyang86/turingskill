# Charles P. Thacker: Selected Writings

## Technical Reports and Documentation

### Alto Hardware Manuals
**Author**: Charles P. Thacker
**Publication**: Xerox PARC Internal Documentation, 1973-1980
**Content**: Complete hardware specifications for the Alto computer, including:
- Processor architecture
- Memory system design
- Display subsystem
- I/O interfaces
- Microcode specifications

**Significance**: These internal documents enabled software development and later system replication.

### Ethernet Implementation Notes
**Author**: Charles P. Thacker
**Publication**: Xerox PARC Technical Reports, 1974-1976
**Content**: Hardware implementation details for Ethernet, including:
- Transceiver design
- Interface specifications
- Signal timing
- Error handling

## Books and Book Chapters

### "Computer Structures: Principles and Examples" (1982)
**Editors**: Daniel P. Siewiorek, C. Gordon Bell, Allen Newell
**Thacker's Contribution**: Chapter on Alto hardware (with Lampson)
**Significance**: Standard computer architecture textbook featuring the Alto as a key example.

### "The Computer Age: A Twenty-Year View" (1979)
**Editors**: Michael L. Dertouzos, Joel Moses
**Thacker's Contribution**: Retrospective and forward-looking essays
**Significance**: Contemporary reflections on computing evolution.

## Turing Award Lecture

### "How the Personal Computer Came to Be"
**Author**: Charles P. Thacker
**Venue**: ACM Turing Award Lecture, 2010
**Published**: Communications of the ACM, 2010

**Key Themes**:
- Origins of the Alto project
- PARC's research culture
- Hardware design philosophy
- Collaboration with Kay, Lampson, Metcalfe
- Evolution of personal computing

**Notable Quote**: "The Alto was built to support a vision of personal computing that Alan Kay had articulated. Our job was to make the hardware that would enable that vision."

## Retrospective Articles

**Significance**: Context for Thacker's contributions from software perspective.
