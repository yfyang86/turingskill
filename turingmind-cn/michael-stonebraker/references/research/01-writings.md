# Michael Stonebraker - Key Writings and Publications

## Landmark Papers

### "The Design and Implementation of INGRES" (1976)
- **Co-authors:** R. Stockton Gaines, et al.
- **Journal:** ACM Transactions on Database Systems
- **Significance:** First relational database implementation
- **Impact:** Launched relational database era

### "The Postgres Next-Generation Database System" (1990)
- **Journal:** Communications of the ACM
- **Innovation:** Object-relational architecture
- **Legacy:** Foundation for PostgreSQL

### "C-Store: A Column-Oriented DBMS" (2005)
- **Co-authors:** MIT students and researchers
- **Conference:** VLDB
- **Impact:** Created column-store database category
- **Commercial:** Vertica founded on this work

### "The End of an Architectural Era" (2007)
- **Conference:** VLDB
- **Argument:** "One size does not fit all"
- **Impact:** Specialized database movement

## Books by Stonebraker

### "Readings in Database Systems" (5th Edition, 2015)
- **Co-editor:** Joseph M. Hellerstein
- **Content:** Curated collection of seminal papers
- **Use:** Standard database course textbook
- **Influence:** Educated generations of database researchers

### "Architecture of a Database System" (2007)
- **Co-authors:** Joseph M. Hellerstein, James Hamilton
- **Publication:** Foundations and Trends in Databases
- **Content:** Comprehensive system architecture overview
- **Impact:** Reference for database implementers

## Technical Articles

### "Operating System Support for Database Management" (1981)
- **Journal:** Communications of the ACM
- **Content:** OS requirements for databases
- **Legacy:** Influenced OS design for databases

### "The Implementation of POSTGRES" (1987)
- **Journal:** IEEE Transactions on Knowledge and Data Engineering
- **Content:** Detailed system implementation
- **Significance:** Practical realization of object-relational ideas
