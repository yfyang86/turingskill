# James H. Wilkinson - Scientific Contributions

## Backward Error Analysis

### Invention and Development
- Co-developed with Leslie Fox and others at NPL
- Revolutionary approach to numerical analysis
- Analyzes computed solution vs. nearby exact problem
- Answers: "What problem did we actually solve?"

### Significance
- Transformed error analysis methodology
- More useful than forward error analysis in practice
- Foundation of modern numerical analysis
- Explains stability of algorithms
- Widely used in numerical linear algebra

### Wilkinson's Principle
- "Backward error analysis is the right way to understand roundoff"
- Algorithms should be backward stable
- Condition number vs. algorithm stability distinction

## Numerical Linear Algebra

### Eigenvalue Problems

### Linear Systems
- Gaussian elimination analysis

### Matrix Computations
- "The Algebraic Eigenvalue Problem" (1965)

## Stability Analysis

### Wilkinson's Polynomial
- Famous example: (x-1)(x-2)...(x-20)
- Shows sensitivity of roots to coefficient perturbations
- Classic teaching example
- Illustrates ill-conditioning

### Algorithm Stability
- Systematic analysis of numerical algorithms
- Stability definitions and criteria
- Error propagation analysis
- Practical guidance for algorithm selection
