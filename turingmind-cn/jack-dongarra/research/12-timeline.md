# Jack Dongarra: Timeline

| Year | Event |
|------|-------|
| 1950 | Born |
| 1972 | BS Mathematics, Chicago State University |
| 1973 | MS Computer Science, Illinois Institute of Technology |
| 1979 | Takes over LINPACK project |
| 1980 | PhD Applied Mathematics, University of New Mexico |
| 1980s | BLAS standardization |
| 1989 | Joins University of Tennessee and Oak Ridge National Lab |
| 1992 | LAPACK released |
| 1993 | Co-founds TOP500 list with Hans Meuer |
| 1995 | ScaLAPACK for distributed memory |
| 2001 | ATLAS autotuning project (Test of Time Award 2016) |
| 2000s | PLASMA for multicore |
| 2007 | Turing Fellow, University of Manchester |
| 2009 | MAGMA for GPU computing |
| 2010s | SLATE for exascale |
| 2017 | NEC C&C Prize |
| 2019 | SIAM/ACM Prize |
| 2021 | **Turing Award** |
| Present | Active research at UT, ORNL, Manchester |

## Career Periods

### Education (1972-1980)
- Mathematics and computer science
- PhD in applied mathematics
- Focus on numerical linear algebra

### Early Career (1980-1989)
- Argonne National Laboratory
- LINPACK leadership
- BLAS development

### Tennessee/ORNL Era (1989-present)
- University faculty position
- Access to leading supercomputers
- Top500 co-founder
- Multiple library generations
