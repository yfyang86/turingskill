# John McCarthy - Scientific Contributions

## Artificial Intelligence Founding

### Dartmouth Conference (1956)
- Co-organized with Marvin Minsky, Claude Shannon, Nathaniel Rochester
- Coined term "Artificial Intelligence"
- Defined initial research agenda
- Proposal stated: "every aspect of learning or any other feature of intelligence can in principle be so precisely described that a machine can be made to simulate it"

### AI Research Agenda
- Logical reasoning as core AI paradigm
- Symbolic approach to intelligence
- Formal methods for knowledge representation
- Commonsense reasoning formalization

## LISP Programming Language

### Invention (1958)
- Created at MIT for AI research
- Second oldest high-level language still in use (after FORTRAN)
- List processing foundation
- Symbolic computation paradigm
- Functional programming influence

### Key Innovations
- **S-expressions:** Code as data, data as code
- **Garbage collection:** Automatic memory management
- **Recursion:** Natural expression of algorithms
- **Dynamic typing:** Runtime type flexibility
- **REPL:** Interactive development

### LISP Dialects
- Common Lisp
- Scheme
- Emacs Lisp
- Clojure
- Influenced JavaScript, Python, Ruby

## Time-Sharing Systems

### Early Concepts
- Pioneer of interactive computing
- Proposed time-sharing in 1959
- Multiple users sharing computer resources
- Foundation of modern operating systems
- Conceptual precursor to cloud computing

### Compatible Time-Sharing System (CTSS)
- Influenced by his proposals
- MIT's early time-sharing system
- Enabled interactive programming
- Led to Multics and Unix

## AI Formalisms

### Situation Calculus
- Formalism for reasoning about actions
- Representing change over time
- Planning and robotics applications
- Foundational for automated reasoning

### Circumscription
- Non-monotonic reasoning formalism
- Handling defaults and exceptions
- Common sense reasoning
- Logical AI approach

### Frame Problem
- Formalized the frame problem
- What changes, what stays the same
- Important AI philosophical problem
- Various proposed solutions

## Programming Language Theory

### ALGOL 60
- Participated in design
- Recursive functions in ALGOL
- Language specification influence
- ALGOL's impact on LISP

### Semantics
- Mathematical semantics of programming
- Formal language definition
- Influenced programming language theory

## Internet and Cloud Concepts

### Early Network Ideas
- Envisioned future computing networks
- Utility computing concepts
- Precursor to cloud computing
- 1960s predictions of modern internet

### Computer Chess
- Early chess programming
- Alpha-beta pruning
- Game tree search
- AI game playing foundation
