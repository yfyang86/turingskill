全面的图灵奖得主认知框架库，涵盖 77 位 A.M. 图灵奖得主（1966-2025）。不是传记，不是语录，而是**他们如何思考**。

---

## 🎯 这是什么？

图灵心智是一个结构化知识库，捕捉了计算机领域最伟大头脑的**思维模式**。每位得主都被呈现为一个"认知操作系统"，你可以激活它以从他们的视角分析问题。

**核心洞见：** 图灵奖得主之所以获胜，不是因为他们写了更好的代码，而是因为他们以不同的方式看待问题。这个库提取并系统化了这些心智模型。

### 使用示例

- "**高德纳**会如何设计这个问题的算法？"
- "哪位图灵奖得主的视角适合我当前的挑战？"
- "**肯·汤普森**会对系统架构权衡说什么？"
- "使用**杰弗里·辛顿**的框架来评估这个机器学习方法"
