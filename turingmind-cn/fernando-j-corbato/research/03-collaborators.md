# Fernando J. Corbato: Key Collaborators

## CTSS Team (1961-1963)
Corbato led the CTSS development:
- **Marjorie Merwin-Daggett**: Early team member
- **Robert Daley**: Implementation
- **MIT Computation Center staff**

## Multics Team (1964-1970s)
Corbato led the Multics project with key contributors:

### MIT Team
- **Victor Vyssotsky**: Bell Labs, co-leader
- **Robert Graham**: System design
- **Robert Daley**: File system
- **Jerome Saltzer**: Security, system principles
- **Numerous graduate students**

### Bell Labs
- **Ken Thompson**: Later created Unix (influenced by Multics)
- **Dennis Ritchie**: Later created C and Unix
- **Victor Vyssotsky**: Project leadership

### General Electric
- **GE engineers**: Hardware support
- **Honeywell**: After acquiring GE computer division

## Unix Connection
Thompson and Ritchie worked on Multics before creating Unix:
- Unix name: UNiplexed Information and Computing Service (Multics pun)
- Unix file system: Simplified Multics hierarchical system
- Unix philosophy: Reaction to Multics complexity

## MIT Colleagues
- **Marvin Minsky**: AI Lab interactions
- **John McCarthy**: Time-sharing concepts
- **Jack Dennis**: Early time-sharing work
- **Herb Simon**: Carnegie connection

## Collaboration Style
Corbato's approach:
- **Team leadership**: Coordinating large projects
- **Academic-industry**: MIT-GE-Bell Labs partnership
- **Student mentorship**: Training next generation
- **Open sharing**: Publishing techniques

## Scientific Community
Influence through:
- Multics documentation
