# Fernando J. Corbato: Key Publications

## Foundational Papers

### "An Experimental Time-Sharing System" (1962)
**Authors**: F.J. Corbato, M. Merwin-Daggett, R.C. Daley
**Venue**: SJCC 1962
**Significance**: First major paper on CTSS. Established time-sharing concepts.

### "The Compatible Time-Sharing System" (1963)
**Author**: F.J. Corbato
**Venue**: IEEE Spectrum
**Significance**: Accessible introduction to CTSS.

### "Introduction and Overview of the Multics System" (1965)
**Authors**: F.J. Corbato, V.A. Vyssotsky
**Venue**: AFIPS 1965
**Significance**: First comprehensive Multics description.

### "Multics: The First Seven Years" (1972)
**Authors**: F.J. Corbato, J.H. Saltzer, C.T. Clingen
**Venue**: AFIPS 1972
**Significance**: Retrospective on Multics development.

## Multics Design Papers

### "Multics OS Design" (various)
Multiple papers on specific Multics features:
- File system design

- Virtual memory
- Dynamic linking

## Turing Award Lecture

### "On Building Systems That Will Fail" (1991)
**Author**: F.J. Corbato
**Venue**: ACM Turing Award Lecture, 1991
**Content**: Reflections on system design, complexity, and lessons learned.

**Key insight**: All complex systems will fail; design for failure handling.
