# Fernando J. Corbato: Key Decisions

## Career Decisions

### Join MIT (1950)
Started at Computation Center. Result: 40+ year career; transformed computing.

### Pursue Time-Sharing (1961)
Created CTSS. Result: Proved interactive computing viable.

### Lead Multics (1964)
Ambitious OS project. Result: Influenced all subsequent OSs.

## Technical Decisions

### Password Authentication
Invented for CTSS. Result: Global standard (with security tradeoffs).

### Hierarchical File System
Multics design. Result: Standard in all OSs.

### Ring-Based Security
Protection rings. Result: x86 architecture feature.

## Collaboration Decisions

### MIT-GE-Bell Labs Partnership
Multics collaboration. Result: Large-scale academic-industry project.

### Open Publication
Shared Multics designs. Result: Widespread influence.

## Decision Style
