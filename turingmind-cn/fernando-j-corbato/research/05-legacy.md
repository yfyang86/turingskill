# Fernando J. Corbato: Legacy and Impact

## Time-Sharing Pioneer
Corbato made computing interactive:
- Before: Batch processing, waits for results
- After: Interactive sessions, immediate feedback
- Impact: How we all use computers today

## Multics Influence
Multics concepts in every modern OS:
- **Hierarchical file system**: Unix, Linux, Windows, macOS
- **Virtual memory**: Standard feature
- **Dynamic linking**: Universal practice
- **Ring protection**: x86 architecture
- **Security model**: Access control foundation

## Unix Lineage
Ken Thompson and Dennis Ritchie:
- Worked on Multics before Unix
- Unix as "simplified Multics"
- Unix name puns on Multics
- Unix concepts trace to Multics

All modern OSs descend from this lineage.

## Password Paradigm
Corbato invented password authentication:
- Ubiquitous in all systems
- Foundation of access control
- Security challenges acknowledged
- Still evolving (biometrics, 2FA)

## Recognition

- National Academy of Engineering
- IEEE, ACM Fellow
- Honorary degrees

## Historical Significance
Corbato is recognized as:

## Long-Term Impact
CTSS and Multics influence continues:
