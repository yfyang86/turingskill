# Dana S. Scott: Timeline

| Year | Event |
|------|-------|
| 1932 | Born October 11, Berkeley, California |
| 1950 | Westinghouse Science Talent Search |
| 1954 | BA Mathematics, UC Berkeley |
| 1958 | PhD Mathematics, Princeton (advisor: Alonzo Church) |
| 1958 | Joins University of Chicago |
| 1960 | Returns to UC Berkeley |
| 1963 | Moves to Stanford |
| 1967 | Steele Prize (continuum hypothesis) |
| 1969 | Domain theory breakthrough |
| 1972 | Joins Oxford as first Professor of Mathematical Logic |
| 1975 | Strachey dies; collaboration ends |
| 1976 | ACM Turing Award (with Rabin) |
| 1981 | Joins Carnegie Mellon |
| 1989 | Hillman Professor |
| 1997 | Rolf Schock Prize |
| 2003 | Retires; returns to Berkeley |
| 2007 | EATCS Award |
| 2009 | Russian Academy Gold Medal |

## Research Periods
- 1957-1960s: Automata and set theory
- 1967-1976: Domain theory and semantics
- 1970s-1980s: Modal logic
- 1981-2003: CMU: teaching and research
- 2003-present: Retirement research
