# John Backus: Key Publications

## FORTRAN Documentation

### "The FORTRAN Automatic Coding System" (1957)
**Authors**: Backus et al.
**Publication**: Proceedings of the Western Joint Computer Conference
**Significance**: Original description of FORTRAN.

### FORTRAN Programmer's Reference Manual (1957)
IBM documentation that spread FORTRAN to the world.

## BNF Papers

### "The Syntax and Semantics of the Proposed International Algebraic Language" (1959)
**Publication**: Pre-ALGOL 60 description using early BNF.

### ALGOL 60 Report (1960)
**With Peter Naur and committee**
Standardized BNF notation for language specification.

## Turing Award Lecture (1977)

### "Can Programming Be Liberated from the von Neumann Style?" (1978)
**Publication**: Communications of the ACM, 21(8):613-641
**Significance**: Manifesto for functional programming; FP language description.

## Functional Programming

### FP Language Papers (1970s-1980s)
- Description of FP system
- Algebra of programs
- Function-level programming concepts

### FL Project Documentation (1980s)
Internal IBM documentation for successor to FP.
