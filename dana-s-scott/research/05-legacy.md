# Dana S. Scott: Legacy and Impact

Scott-Rabin paper:
- Nondeterminism as fundamental concept
- Basis for regular language theory
- Compiler design applications
- Still taught in every CS curriculum

## Domain Theory Revolution
Scott's deepest impact:
- Mathematical semantics for programming languages
- Foundation for functional programming
- Type theory development
- Program verification methods

- Compiler verification enabled
- Rigorous software engineering
