# Tony Hoare: Legacy and Impact

## Quicksort: Ubiquitous Algorithm
- Default sort in Unix, C, Python, Java
- Foundation of algorithm courses
- Divide-and-conquer paradigm
- 65+ years of continuous use

## Hoare Logic: Verification Foundation
- Taught in every CS program
- Compiler verification methods
- Static analysis tools
- Type system theory

## CSP: Concurrency Theory
- Go language goroutines
- Erlang actor model
- Occam programming
- Formal verification of concurrent systems

## The Billion Dollar Mistake Impact
Honest acknowledgment led to:
- Swift optionals
- Kotlin null safety
- Rust Option<T>
- TypeScript strictNullChecks
- Modern language safety

## Formal Methods Field
Hoare established:
- Program verification as discipline
- Industrial formal methods
- Safety-critical system certification
- Z notation and tools

## Honors
- Turing Award (1980)
- Kyoto Prize (1990)
- Knighthood (2000)
- Royal Society Fellowship
- Multiple medals and awards

## Ongoing Influence
Current relevance:
- Go language (Google)
- Rust safety
- Verified software
- Concurrent systems

## Quotes That Endure
- "There are two ways of constructing a software design..."
- "Billion dollar mistake"
- Influenced thinking about simplicity and correctness

Hoare transformed software from craft to engineering with mathematical rigor.
