# Maurice V. Wilkes - Scientific Contributions

## Stored-Program Computer Development

### EDSAC (Electronic Delay Storage Automatic Calculator)
- **1949:** First operational stored-program computer
- Inspired by John von Neumann's EDVAC report
- 1024 words of memory (mercury delay lines)
- Practical implementation of stored-program concept
- Enabled true general-purpose computing

### Subroutine Libraries
- Invented concept of reusable subroutines
- Created first subroutine library for EDSAC (1949)
- Established conventions for subroutine calling
- Foundation for modern software engineering
- Wheeler Jump (computed GOTO for subroutines)

## Microprogramming

### Invention and Development
- **1951:** Invented microprogramming (published 1955)
- Used on EDSAC 2 to simplify control unit design
- Enabled flexible instruction set implementation
- Made complex architectures practical
- Foundation for modern processor design

### Microprogramming Impact
- Simplified computer design and maintenance
- Enabled commercial mainframe development
- Used in IBM System/360 and successors
- Still relevant in modern processor microcode
- Established computer architecture as discipline

## Computer Architecture

### EDSAC 2 Design
- First computer using microprogramming extensively
- Bit-slice architecture concepts
- Index registers for array processing
- Influenced subsequent British computers

### Titan/Atlas 2
- Multi-programming operating system
- Virtual memory implementation
- Time-sharing capabilities
- Commercial application of academic research

## Software Engineering

### Early Programming Methodology
- Emphasized systematic programming approaches
- Developed debugging techniques
- Created programming documentation standards
- Library concept influenced modular programming

### Operating Systems
- Early work on monitor programs
- Input/output channel concepts
- Multi-programming foundations
- Resource management techniques

## Educational Contributions

### Cambridge Computer Laboratory
- Established premier computer science institution
- Trained generation of British computer scientists
- Created model for academic computer science
- International influence through students and visitors

### Textbooks and Writing
- "The Preparation of Programs for an Electronic Digital Computer" (1951)
- "Automatic Digital Computers" (1956)
- "Time-Sharing Computer Systems" (1968)
- Clear, practical writing style

## Legacy Impact

- Microprogramming universal in processor design
- Stored-program concept universally adopted
- Subroutine libraries standard in all programming
- Cambridge model copied worldwide
- Bridge between academic and commercial computing
