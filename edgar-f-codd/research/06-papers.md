# Edgar F. Codd - Key Publications

## Foundational Papers

### A Relational Model of Data for Large Shared Data Banks (1970)
- **Journal**: Communications of the ACM, Vol. 13, No. 6
- **Pages**: 377-387
- **Impact**: The seminal paper defining relational database theory
- **Key Contributions**:
  - Mathematical foundation based on predicate logic
  - Introduction of relations, tuples, and attributes
  - Relational algebra operations
  - Data independence concept
- **Citations**: 15,000+ (most cited database paper)

### Further Normalization of the Data Base Relational Model (1971)
- **Publication**: Courant Computer Science Symposia Series, Vol. 6
- **Editor**: Randall Rustin
- **Publisher**: Prentice-Hall
- **Content**: Second and Third Normal Forms formalized
- **Significance**: Established normalization theory

### Relational Completeness of Data Base Sublanguages (1972)
- **Publication**: Courant Computer Science Symposia Series, Vol. 6
- **Content**: Relational calculus completeness
- **Impact**: Theoretical foundation for query languages
- **Result**: Codd's Theorem on equivalence of relational algebra and calculus

## Query Language Design

### SEQUEL: A Structured English Query Language (1974)
- **Authors**: Donald D. Chamberlin, Raymond F. Boyce
- **Acknowledgment**: Based on Codd's relational model
- **Conference**: ACM SIGFIDET Workshop
- **Evolution**: Became SQL
- **Legacy**: Most widely used database language

### Data Base Sublanguages (1971)
- **Publication**: Courant Computer Science Symposia Series
- **Content**: Alpha query language specification
- **Significance**: Early declarative query language design

## System R Papers

### System R: An Architectural Overview (1976)
- **Authors**: Astrahan et al. (Codd research group)
- **Journal**: IBM Research Report RJ2081
- **Content**: First industrial-strength relational system
- **Impact**: Proved practical viability

### System R: Relational Approach to Database Management (1976)
- **Journal**: ACM Transactions on Database Systems
- **Content**: Query optimization, locking, recovery
- **Significance**: Foundation for IBM DB2

## Later Theoretical Work

### Extending the Database Relational Model to Capture More Meaning (1979)
- **Journal**: ACM Transactions on Database Systems, Vol. 4, No. 4
- **Pages**: 397-434
- **Content**: RM/T (Relational Model/Tasmania)
- **Features**: Entity integrity, referential integrity
- **Impact**: Extended relational theory

### The Twelve Rules for Relational Database Systems (1985)
- **Publication**: ComputerWorld
- **Content**: Criteria for truly relational DBMS
- **Purpose**: Combat misleading vendor claims
- **Legacy**: Benchmark for database evaluation

### Relational Database: A Practical Foundation for Productivity (1982)
- **Journal**: Communications of the ACM, Vol. 25, No. 2
- **Content**: Turing Award lecture
- **Significance**: Comprehensive retrospective
- **Pages**: 109-117
