# John Backus: Legacy and Impact

## FORTRAN's Enduring Influence
- First widely-used high-level language
- Still used in scientific computing (HPC, weather, physics)
- Model for subsequent languages
- Proved compilers could optimize

## BNF as Universal Standard
- Every programming language specification
- Compiler construction courses
- Parser generators
- Formal language theory

## Functional Programming Movement
Turing Award lecture sparked:
- Modern functional languages (Haskell, ML, Scala)
- Features in mainstream languages (Java streams, Python lambdas)
- Declarative programming paradigm
- Parallel and concurrent programming approaches

## Programming Paradigm Shifts
Backus enabled:
- Machine-independent programming
- Abstraction from hardware
- Programmer productivity focus
- Mathematical approaches to programs

## Honors and Recognition
- IBM Fellow (1963)
- National Medal of Science (1975)
- Turing Award (1977)
- Draper Prize (1993)
- Computer History Museum Fellow (1997)
- Asteroid named after him (2007)

## Quotes That Live On
- "Much of my work has come from being lazy"
- "You need the willingness to fail all the time"
- "Can Programming Be Liberated from the von Neumann Style?"

## Daily Impact
Every programmer today uses Backus's contributions:
- High-level languages
- Compilers and optimization
- Formal syntax
- Functional programming concepts

His work transformed programming from craft to engineering discipline.
