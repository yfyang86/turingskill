# James H. Wilkinson - Legacy and Impact

## Scientific Legacy

### Backward Error Analysis
- **Universal Methodology:** Standard in numerical analysis
- **Every Numerical Algorithm:** Analyzed using his approach
- **Software Development:** Stability criterion essential
- **Condition Numbers:** Standard tools in all scientific computing

### Numerical Linear Algebra
- **Eigenvalue Algorithms:** QR algorithm variants still used
- **LAPACK Foundation:** Direct intellectual descendant
- **MATLAB Implementation:** His algorithms in core functions
- **SciPy/NumPy:** Built on his theoretical work

### Stability Analysis
- **Wilkinson's Polynomial:** Classic teaching example
- **Algorithm Selection:** Stability criteria standard
- **Error Bounds:** Required in numerical software
- **Ill-Conditioning:** Understanding through his work

## Institutional Legacy

### National Physical Laboratory
- Built world-class numerical analysis group
- NPL remained center for numerical analysis
- Government research model
- International reputation

### Numerical Analysis Community
- IFIP Working Group 2.5 founding

- Conference culture establishment
- International cooperation model

## Educational Legacy

### Textbooks
- **"Rounding Errors in Algebraic Processes"** (1963)
  - Classic monograph, still referenced
  - Backward error analysis foundation

- **"The Algebraic Eigenvalue Problem"** (1965)
  - Definitive eigenvalue reference
  - Still taught in graduate courses
  - Mathematical elegance with practicality

- **"Handbook for Automatic Computation, Vol. 2"** (1971)
  - With C. Reinsch
  - Standard numerical software reference

### Teaching Model
- Clear exposition
- Theory with practical computation
- Rigorous yet accessible
- Bridge mathematics and computing
