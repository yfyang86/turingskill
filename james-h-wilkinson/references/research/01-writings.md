# James H. Wilkinson - Writings

## Original Publications

### Books Authored
- **"Rounding Errors in Algebraic Processes"** (1963)
  - Prentice-Hall and HMSO
  - Classic monograph on error analysis
  - Backward error analysis foundation
  - Still referenced in courses

- **"The Algebraic Eigenvalue Problem"** (1965)
  - Oxford University Press
  - 662 pages, definitive eigenvalue reference
  - Standard graduate text
  - Mathematical rigor with practicality

- **"Handbook for Automatic Computation, Vol. II: Linear Algebra"** (1971)
  - With C. Reinsch
  - Springer-Verlag
  - ALGOL algorithms
  - Software standard
