# Ole-Johan Dahl - Key Papers and Publications

## Foundational Papers

### Simula Language Design
- **"SIMULA - An ALGOL-Based Simulation Language"** (1966)
  - With Kristen Nygaard
  - Communications of the ACM
  - Introduced Simula I concepts
  - Established simulation language paradigm

- **"The Development of the SIMULA Languages"** (1978)
  - With Kristen Nygaard
  - History of Programming Languages Conference
  - Definitive historical account
  - Design decisions and rationale

### Object-Oriented Programming Origins
- **"Class and Subclass Declarations"** (1970)
  - IFIP Working Conference
  - Formalized inheritance concept
  - Implementation details

- **"Structured Programming"** (1972)
  - With Dijkstra, Hoare (chapter contributor)
  - Academic Press
  - Influential textbook
  - Formal methods chapter

## Books and Monographs

### Textbooks
- **"Structured Programming"** (1972)
  - Co-authored with Edsger Dijkstra and C.A.R. Hoare
  - Classic programming methodology text
  - Dahl contributed chapter on program construction

- **"Verifiable Programming"** (1992)
  - Solo work on formal verification
  - Prentice-Hall International
  - Comprehensive treatment of verification
  - Mathematical approach to correctness

### Language Documentation
- **Simula 67 Common Base Language** (1968, revised)
  - With Nygaard and others
  - Language standard document
  - Implementation guide
  - Used by all Simula implementers
