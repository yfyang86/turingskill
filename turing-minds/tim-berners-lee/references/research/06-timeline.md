# Tim Berners-Lee: Timeline

## Early Life and Education

| Year | Event |
|------|-------|
| 1955 | Born June 8, London, England |
| 1960s | Childhood exposure to computers through parents' work on Ferranti Mark I |
| 1973 | Enters Queen's College, Oxford to study Physics |
| 1976 | Graduates with first-class Physics degree; builds computer from TV and microprocessor |

## Early Career

| Year | Event |
|------|-------|
| 1976 | Joins Plessey Telecommunications as software engineer |
| 1978 | Moves to D.G. Nash; develops typesetting software |
| 1980 | First contract work at CERN; develops ENQUIRE hypertext system |
| 1981-1984 | Works at Image Computer Systems; builds software for printers |

## Creation of the World Wide Web

| Year | Event |
|------|-------|
| 1984 | Returns to CERN as Fellow; works on distributed real-time systems |
| March 1989 | Submits "Information Management: A Proposal" to CERN |
| 1990 | Receives approval; begins development on NeXT computer |
| Dec 1990 | Completes first web browser (WorldWideWeb) and server (httpd) |
| Aug 1991 | First website (info.cern.ch) goes public |
| 1991 | Nicola Pellow develops Line Mode Browser |
| 1993 | CERN releases web technology to public domain (April 30) |
| 1993 | Marc Andreessen releases Mosaic browser |

## Institutionalization and Growth

| Year | Event |
|------|-------|
| 1994 | Joins MIT Laboratory for Computer Science |
| 1994 | Founds World Wide Web Consortium (W3C) |
| 1995 | Microsoft embraces the web with Internet Explorer |
| 1996 | W3C establishes formal standards process |

## Recognition and Expansion

| Year | Event |
|------|-------|
| 1999 | Publishes autobiography "Weaving the Web" |
| 2001 | Publishes "The Semantic Web" article in Scientific American |
| 2004 | Knighted by Queen Elizabeth II (Sir Tim) |
| 2004 | Appointed Professor at University of Southampton |
| 2006 | Creates Linked Data principles |
| 2007 | Appointed to Order of Merit; first recipient of the Millennium Technology Prize |

## Mature Advocacy

| Year | Event |
|------|-------|
| 2009 | Delivers TED talk on Linked Data |
| 2009 | Launches World Wide Web Foundation |
| 2012 | Olympics opening ceremony appearance ("This is for everyone") |
| 2013 | Reveals concerns about surveillance post-Snowden |
| 2016 | Receives Turing Award |
| 2016 | Launches Solid project at MIT |
| 2017 | Appointed Professor at Oxford University |
| 2018 | Publishes article on web's "three challenges" |
| 2019 | Launches "Contract for the Web" |

## Current Era

| Year | Event |
|------|-------|
| 2020 | Solid project commercialization through Inrupt |
| 2021 | Continued advocacy for data sovereignty |
| 2023 | BBC Reith Lectures on web's future |
| 2024 | Ongoing development of Solid specifications |
| Present | Active at MIT CSAIL, Oxford, and as web advocate |

## Key Milestones Summary

- **1989**: Web concept proposed
- **1991**: First website live
- **1993**: Technology released freely
- **1994**: W3C founded
- **2004**: Knighted
- **2016**: Turing Award and Solid launch
- **2019**: Contract for the Web

The timeline reveals a consistent pattern: invention (1989-1993), institutionalization (1994-2004), and ongoing stewardship (2004-present) with increasing focus on ethical challenges.
