# Tim Berners-Lee: Key Publications

## Foundational Document

### "Information Management: A Proposal" (1989)

This seminal document, submitted to CERN management, proposed a distributed hypertext system for information sharing. Written in March 1989, the proposal outlined what would become the World Wide Web. Though initially dismissed as "vague but exciting," it contained the core concepts of hypertext, distributed information, and universal access. The paper's humble origins—merely an internal CERN document—belies its world-changing impact.

## Technical Specifications

### The Original HTML Tags (1991)

Berners-Lee's initial HTML specification defined 18 tags, including foundational elements like `<title>`, `<a>` (anchor), and `<p>` (paragraph). This simplicity enabled rapid adoption while extensibility allowed evolution.

### HTTP Specifications

Berners-Lee authored early HTTP protocol definitions, establishing the request-response model that underlies web communication. The protocol's stateless design proved both a feature (scalability) and limitation (session management).

### URI/URL Specification

His work on Uniform Resource Identifiers created the addressing system enabling global web navigation. RFC 1630 and subsequent standards established the syntax still used today.

## W3C Technical Reports

As W3C Director, Berners-Lee oversaw hundreds of specifications, with personal contributions to:
- **HTML 4.0 Specification** (1997)
- **XML 1.0 Specification** (1998)
- **RDF Model and Syntax** (1999)
- **OWL Web Ontology Language** (2004)

## Semantic Web Papers

### "The Semantic Web" (2001)

Co-authored with James Hendler and Ora Lassila in Scientific American, this article introduced the Semantic Web vision to a broad audience. It described a web where machines could process information with understanding, enabling intelligent agents and automated reasoning.

### Linked Data Publications

Berners-Lee's 2006 note on Linked Data established four principles for publishing structured data:
1. Use URIs as names for things
2. Use HTTP URIs
3. Provide useful information when URIs are looked up
4. Include links to other URIs

## Books

### "Weaving the Web" (1999)

Berners-Lee's autobiography chronicles the web's creation from personal perspective. It reveals technical decisions, philosophical motivations, and his vision for the web's future. The book provides essential primary source material for understanding web history.

### Other Publications

He has authored numerous articles in scientific journals, conference proceedings, and popular media, consistently advocating for an open, accessible web.

## Recent Work

Berners-Lee continues publishing on Solid, data sovereignty, and web governance, maintaining active scholarly engagement with the technology he created.
