# Shafi Goldwasser: Legacy and Impact

## Transforming Cryptography
Goldwasser (with Micali) made cryptography a rigorous science:
- Before: Ad hoc designs, heuristic security
- After: Formal definitions, provable security
- Impact: Modern secure systems

## Key Innovations

### Zero-Knowledge Proofs
- Fundamental concept in cryptography
- Enabled blockchain privacy (zk-SNARKs)
- Anonymous credentials
- Secure voting

### Probabilistic Encryption
- Semantic security standard
- Influenced all modern encryption
- RSA-OAEP, ElGamal, etc.

### Interactive Proofs
- Led to PCP theorem
- Complexity theory advances
- Probabilistic verification
