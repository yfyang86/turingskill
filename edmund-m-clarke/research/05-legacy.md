# Edmund M. Clarke - ulegacy

## Overview
Edmund Melson Clarke was an American computer scientist and 2007 Turing Award winner for model checking.

## Turing Award 2007
- Shared with E. Allen Emerson and Joseph Sifakis
- For developing Model-Checking into a highly effective verification technology

## Key Information
- Born: July 27, 1945, Newport News, Virginia
- Died: December 22, 2020
- Professor at Carnegie Mellon University
- PhD: Cornell University

## Major Contributions
- Model checking
- SMV model checker
- Temporal logic verification
- Hardware and software verification
- Counterexample-guided abstraction

## Research Focus
- Formal verification
- Model checking algorithms
- Temporal logic
- Hardware verification
- Software verification
