# Ole-Johan Dahl - Legacy and Impact

## Object-Oriented Programming Revolution

### Paradigm Creation
- Co-created OOP with Kristen Nygaard
- Fundamental shift in software development
- From procedure-oriented to object-oriented
- Changed how we think about programming

### Conceptual Innovations
- Class and object concepts
- Inheritance mechanism
- Encapsulation and information hiding
- Message passing paradigm

## Influence on Programming Languages

### Direct Lineage
- **Simula 67:** First OOP language
- **Smalltalk:** Pure OOP evolution
- **C++:** Bjarne Stroustrup directly inspired
- **Java:** Mainstream OOP adoption
- **C#, Python, Ruby:** All owe debt to Simula

### Language Features
- Classes as user-defined types
- Objects with state and behavior
- Inheritance for code reuse
- Polymorphism through virtual procedures
