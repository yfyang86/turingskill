# Jack Dongarra: Key Publications

## Foundational Software Libraries

### LINPACK Users' Guide (1979)
**Authors**: J.J. Dongarra, J.R. Bunch, C.B. Moler, G.W. Stewart
**Publisher**: SIAM
**Significance**: Documented the LINPACK linear algebra library
**Impact**: Became standard reference for numerical linear algebra

### LAPACK Users' Guide (1992, multiple editions)
**Authors**: E. Anderson, Z. Bai, C. Bischof, S. Blackford, J. Demmel, J. Dongarra, et al.
**Publisher**: SIAM
**Significance**: Standard reference for modern linear algebra
**Impact**: Still the primary reference 30+ years later

## Key Research Papers

### "An Extended Set of FORTRAN Basic Linear Algebra Subprograms" (1988)
**Authors**: J. Dongarra, J. Du Croz, S. Hammarling, R. Hanson
**Publication**: ACM Trans. Math. Software
**Significance**: Defined Level 2 BLAS standard
**Impact**: Foundation for optimized linear algebra

### "A Set of Level 3 Basic Linear Algebra Subprograms" (1990)
**Authors**: J. Dongarra, J. Du Croz, I. Duff, S. Hammarling
**Publication**: ACM Trans. Math. Software
**Significance**: Defined Level 3 BLAS (matrix-matrix operations)
**Impact**: Enabled cache-efficient algorithms

### "A Message Passing Interface Standard" (1994)
**Authors**: MPI Forum (including Dongarra)
**Significance**: Established MPI as parallel computing standard
**Impact**: Universal standard for distributed memory computing

### "Automatically Tuned Linear Algebra Software (ATLAS)" (2001)
**Authors**: C. Whaley, A. Petitet, J. Dongarra
**Publication**: SC2001
**Significance**: Automatic optimization of BLAS
**Impact**: 2016 Test of Time Award

### "Performance, Design, and Autotuning of Batched GEMM for GPUs" (2016)
**Authors**: Various, led by Dongarra group
**Publication**: International Conference on High Performance Computing
**Significance**: Optimized many small matrix operations
**Impact**: Critical for machine learning

### "MAGMA: A New Generation of Linear Algebra Library for GPU and Multicore Architectures" (2010s)
**Authors**: Dongarra, Tomov, Nath, and team
**Significance**: GPU-accelerated linear algebra
**Impact**: Enables modern AI and scientific computing

## TOP500 Reports

### TOP500 Supercomputer Sites (Biannual, 1993-present)
**Authors**: Dongarra, Meuer, Strohmaier, Simon, et al.
**Publication**: Published twice yearly at SC and ISC conferences
**Significance**: Definitive ranking of supercomputers
**Impact**: Drives HPC technology development

## Books

### *Numerical Linear Algebra for High-Performance Computers* (1998)
**Authors**: Dongarra, Duff, Sorensen, van der Vorst
**Publisher**: SIAM
**Significance**: HPC-focused linear algebra textbook

### *Introduction to High-Performance Computing for Scientists and Engineers* (2010)
**Co-authors**: Various
**Significance**: Educational resource for HPC

## Citation Impact

Dongarra's papers are among the most cited in scientific computing:
- BLAS papers: thousands of citations each
- LAPACK reference: standard citation in numerical computing
- TOP500 reports: widely referenced
- Software itself is cited more than papers

## Software as Publication

Dongarra's primary contribution is software, not traditional papers:
- Millions of users of his libraries
- Software runs on billions of devices
- Impact exceeds any paper citation count
- Open source enables inspection and verification

## Publication Philosophy

Dongarra's approach:
- Software is the primary output
- Papers document and explain algorithms
- Open source enables reproducibility
- Standards create lasting impact
