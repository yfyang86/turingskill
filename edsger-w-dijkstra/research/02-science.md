# Edsger W. Dijkstra - Scientific Contributions

## Algorithms

### Dijkstra's Algorithm (1956)
- Shortest path in weighted graphs
- Greedy algorithm paradigm
- O(V²) time complexity (improved to O(E + V log V))
- Used in GPS navigation, network routing
- Published in 1959

### Shunting-yard Algorithm
- Infix to postfix conversion
- Expression parsing
- Stack-based algorithm
- Used in compilers and calculators

## Concurrent Programming

### Semaphores (1965)
- Synchronization primitive
- P() and V() operations
- Mutual exclusion mechanism
- Solution to dining philosophers problem
- Foundation of concurrent programming

### THE Operating System (1968)
- Seminal operating system design
- Structured in layers
- First demonstration of structured OS
- Semaphore use throughout
- Verified to be deadlock-free

## Structured Programming

### GOTO Statement Considered Harmful (1968)
- Letter to Communications of the ACM
- Sparked structured programming movement
- Advocated for control structure discipline
- Influenced language design (Pascal, C, etc.)
- Still debated but widely adopted

### Structured Programming Principles
- Single entry, single exit
- Control flow discipline
- Abstraction and modularity
- Top-down design
- Influenced software engineering

## Formal Methods

### Predicate Transformer Semantics
- Weakest precondition calculus
- Formal program verification
- wp(S, Q) - weakest precondition
- "A Discipline of Programming" (1976)
- Rigorous approach to correctness

### Program Correctness
- Hoare logic contributions
- Formal specification
- Verification methodology
- Mathematical rigor in programming

## Language Design

### ALGOL 60
- Implementer on Electrologica X1
- Report editor
- Language design influence
- Block structure advocacy

### Guarded Commands
- Non-deterministic constructs
- if-fi and do-od
- "Guarded Commands, Nondeterminacy and Formal Derivation" (1975)
- Influenced Ada, CSP, Occam
