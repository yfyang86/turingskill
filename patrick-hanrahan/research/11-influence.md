# Patrick Hanrahan - Influence on the Field

## Impact on Computer Graphics

### Shading Languages
- RenderMan Shading Language (RSL) was first widely-used shading language
- Directly influenced:
  - OpenGL shading language (GLSL)
  - HLSL (Microsoft DirectX)
  - Cg (NVIDIA)
  - Modern GPU shader models
- Established paradigm of programmable materials

### Volume Rendering
- Algorithms became standard for medical visualization
- CT and MRI data visualization in healthcare
- Scientific visualization across disciplines
- Hardware acceleration techniques adopted industry-wide

## Influence on GPU Computing

### BrookGPU
- First high-level language for GPU programming
- Precursor to CUDA
- Demonstrated general-purpose computing on GPUs
- Influenced parallel programming models

### GPU Architecture
- Consultation with NVIDIA on programmable pipelines
- Shading language concepts in hardware design
- Stream processing model adoption

## Data Visualization Revolution

### Tableau's Impact
- Created self-service business intelligence category
- Influenced design of:
  - Microsoft Power BI
  - Google Data Studio
  - Amazon QuickSight
  - Countless other tools
- Changed how businesses use data

### Visual Analytics Field
- Established academic discipline
- Influenced information visualization research
- Business adoption of data-driven decision making
