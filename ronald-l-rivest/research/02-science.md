# Ronald L. Rivest - Scientific Contributions

## RSA Cryptosystem

### The Problem
Need for secure communication without shared secret keys

### The Solution (1977)
With Shamir and Adleman:
- Public-key encryption using integer factorization
- Public key for encryption
- Private key for decryption
- Mathematical asymmetry enables security

### Technical Details
- Based on difficulty of factoring large integers
- Euler's theorem foundation
- Practical key sizes (512+ bits originally, 2048+ now)
- Efficient algorithms for encryption/decryption

### Security Foundation
- Computational hardness of factoring
- No known polynomial-time algorithm
- Quantum threat (Shor's algorithm)
- Post-quantum cryptography research

## Hash Functions

### MD5 (1991)
- 128-bit hash function
- Widely used for checksums
- Cryptographic weaknesses discovered later
- Still used for non-cryptographic purposes

### RC Algorithms
- RC2, RC4, RC5, RC6
- Variable key-size stream and block ciphers
- RC4 widely used in SSL/TLS (now deprecated)
- Simple, efficient designs
