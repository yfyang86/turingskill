# Kristen Nygaard - Key Papers and Publications

## Foundational Papers

### Simula Language Design
- **"SIMULA - An ALGOL-Based Simulation Language"** (1966)
  - With Ole-Johan Dahl
  - Communications of the ACM
  - Introduced Simula I
  - Established simulation language paradigm

- **"The Development of the SIMULA Languages"** (1978)
  - With Ole-Johan Dahl
  - HOPL Conference
  - Historical and technical account
  - Design rationale documentation

### Object-Oriented Programming
- **"Class and Subclass Declarations"** (1970)
  - IFIP Working Conference
  - Formalized inheritance concept
  - Implementation approach
