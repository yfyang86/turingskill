# Kristen Nygaard - Scientific Contributions

## Object-Oriented Programming Origins

### The Vision: Modeling Reality
Nygaard approached programming from need to model complex systems:
- Real-world systems have interacting components
- Components maintain state over time
- Need for abstraction matching human cognition
- Simulation as paradigm for computation

### Conceptual Innovations
- **Object concept:** Entity with state and behavior
- **Class concept:** Template for object creation
- **Inheritance:** Hierarchical classification
- **Encapsulation:** Information hiding

## Simula Language Development

### Simula I (1962-1964)
**Purpose:** Discrete event simulation

Nygaard's contributions:
- Process concept for active objects
- Simulation worldview
- Real-world system modeling
- User-centered language design

### Simula 67 (1965-1967)
**Evolution:** General-purpose language

Key extensions:
- Unified class mechanism
- Prefix classes (inheritance)
- Virtual procedures
- Reference semantics
