# Tim Berners-Lee: Writings and Documentation

## Primary Sources

### Foundational Documents
- **"Information Management: A Proposal"** (March 1989)
  - Internal CERN document proposing the World Wide Web
  - Available at: w3.org/History/1989/proposal.html
  - The genesis document of the modern web

- **The Original WWW FAQ**
  - Early documentation explaining web concepts
  - Demonstrates Berners-Lee's commitment to clear communication

### Technical Specifications
- HTML Tags (1991-1993)
- HTTP Protocol drafts (1991-1996)
- URI/URL specifications
- All archived at W3C Technical Reports

## Books by Berners-Lee

### "Weaving the Web" (1999, Harper Collins)
- The definitive autobiography
- ISBN: 978-0062515872
- Covers the web's creation through the dot-com era
- Reveals personal philosophy and decision-making process

### "A Framework for Web Science" (2006)
- Co-authored with Wendy Hall and James Hendler
- Academic treatment of web as scientific object of study
- Establishes web science as interdisciplinary field

## Academic Papers

### Foundational Academic Works
- "The Semantic Web" (Scientific American, May 2001)
  - Co-authored with Hendler and Lassila
  - Introduced Semantic Web to general audience

- "Linked Data" (2006, W3C Design Issues)
  - Defining document for Linked Data principles
  - Practical guide for data publishers

### Recent Publications (2016-Present)
- Solid project documentation
- "The Web We Want" series of essays
- Articles on data sovereignty and decentralization
- Published in Nature, Communications of the ACM, and Scientific American
