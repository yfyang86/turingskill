# Stephen A. Cook - Legacy and Impact

## Immediate Impact (1970s)

**Theoretical Computer Science**
- Established complexity theory as core discipline
- NP-completeness became central concept
- STOC/FOCS conferences gained prominence
- Graduate curriculum transformation

**Algorithm Design Revolution**
- Designers now classify problem difficulty
- Heuristic approaches for NP-hard problems
- Approximation algorithm development
- Average-case analysis emergence

**Cryptography Foundations**
- Security based on hardness assumptions
- One-way functions importance
- Reductionist security proofs
- Modern cryptography development

**Industry Applications**

## NP-Completeness Catalog

**Karp's 21 Problems (1972)**
- Cook provided framework
- Karp populated the landscape
- Now thousands of problems classified
- Garey & Johnson textbook (1979)

**Problem Categories**
- Graph theory: Clique, coloring, Hamiltonian cycle
- Optimization: Knapsack, bin packing, TSP
- Logic: SAT, QBF, constraint satisfaction
- Number theory: Factoring, discrete log (different complexity)

**Approximation Results**
- Some NP-hard problems approximable
- Others have hardness of approximation
- PCP theorem (Arora, Safra, 1998)
- Optimal approximation ratios known

## Proof Complexity Legacy

**Cook-Reckhow Program**
- Propositional proof systems classification
- Lower bounds as approach to P vs NP
- Superpolynomial lower bounds proven
- Connections to bounded arithmetic

**Major Results**
- Haken's resolution lower bounds (1985)
- Pudlák's work on Frege systems
- Algebraic proof systems (IPS)
- Geometric proof complexity

**Open Problems**
- Superpolynomial Frege lower bounds
- Extended Frege separation
- Propositional vs. predicate logic
- Connections to circuit complexity
