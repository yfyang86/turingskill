A comprehensive cognitive framework library for all 77 A.M. Turing Award laureates (1966-2025). Not biographies. Not quotes. **How they think.**

---

## 🎯 What is This?

Turing Minds is a structured knowledge base that captures the **thinking patterns** of computing's greatest minds. Each laureate is represented as a "cognitive operating system" that you can activate to analyze problems from their perspective.

**The core insight:** Turing Award winners didn't just write better code—they saw problems differently. This library extracts and systematizes those mental models.

### Example Use Cases

- "How would **Donald Knuth** approach algorithm design for this problem?"
- "Which Turing laureate's perspective fits my current challenge?"
- "What would **Ken Thompson** say about system architecture tradeoffs?"
- "Use **Geoffrey Hinton's** framework to evaluate this ML approach"

## 📖 File Types Explained

### Core Framework (`SKILL.md`)
- Role-playing rules for authentic perspective-taking
- 4 core mental models with evidence & limitations
- 6-7 decision heuristics
- Expression DNA (communication style)
- Values, anti-patterns, and intellectual tensions

### Research Files (`research/`)

| File | Purpose |
|------|---------|
| `01-biography.md` | Birth, education, career trajectory, personal influences |
| `02-science.md` | Technical contributions with timelines and breakthrough moments |
| `03-collaborators.md` | Key partnerships (Thompson+Ritchie, Deep Learning trio) |
| `04-ethics.md` | Views on technology ethics, warnings, controversies |
| `05-legacy.md` | Lasting impact, students influenced, fields shaped |
| `06-papers.md` | Annotated bibliography of most important works |

### Reference Files (`references/research/`)

| File | Purpose |
|------|---------|

| `04-external-views.md` | Tributes, criticisms, peer assessments |
| `05-decisions.md` | Analysis of key career/technical decisions |
| `06-timeline.md` | Detailed chronological events |

## 🎭 Thinking Styles

Each laureate embodies distinct cognitive patterns:

| Style | Laureates | Keywords |
|-------|-----------|----------|
| 🔥 **Contrarian Persistence** | Hinton, Bachman, Dijkstra | Against the mainstream, unwavering |
| 🔬 **Extreme Observation** | Hamming, Minsky, Sutherland | Seeing what others miss |
| 🧩 **Interdisciplinary** | Simon/Newell, Sutton/Barto, Rabin | Cross-domain synthesis |
| ⏳ **Long-term Deep Work** | Knuth, Iverson, Sutherland | Decades on one problem |
| ⚡ **Technical Breakthrough** | Cook, Karp, Deep Learning trio | New methods enable discovery |
| 🤝 **Collaborative** | Thompson/Ritchie, Cerf/Kahn | Shared vision, complementary skills |
| 📐 **Systems Thinking** | Lamport, Liskov, Brooks | Complexity, architecture |
