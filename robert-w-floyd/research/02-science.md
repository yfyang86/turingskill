# Robert W. Floyd - Scientific Contributions

## Algorithms

### Floyd's Algorithm (1962)
- All-pairs shortest path
- Dynamic programming approach
- O(V³) time complexity
- Uses adjacency matrix representation
- Foundation for many graph algorithms

### Floyd-Warshall Algorithm
- Efficient transitive closure
- Boolean matrix operations
- Dynamic programming
- Widely used in applications
- Various optimizations developed

### HeapSort (1964)
- In-place sorting algorithm
- O(n log n) guaranteed
- No extra space required
- Tree-based heap structure
- Elegant and practical

## Program Verification

### Floyd-Hoare Logic
- **"Assigning Meanings to Programs"** (1967)
  - Seminal paper in program verification
  - Assertion-based verification
  - Pre- and post-conditions
  - Foundation of formal methods

- **Inductive Assertions**
  - Loop invariants
  - Verification conditions
  - Mathematical rigor
  - Hoare built on his work

### Impact on Verification
- Formal methods foundation
- Software engineering discipline
- Critical systems verification
- Hoare logic evolution
