# 02 - 核心科学贡献

## 时间线

### 1964年：鼠标发明
**发明**：计算机鼠标
**动机**：替代键盘的光标控制
- 木质外壳，两个轮子
- 专利授权给Apple免费使用
- 革命性的输入设备

### 1968年："所有演示之母"
**事件**：Fall Joint Computer Conference演示
**展示**：
- 鼠标操作
- 超文本链接
- 视频会议
- 协同编辑
- 图形界面

**历史意义**：
- 展示了未来计算的可能性
- 影响了Apple、Xerox PARC、Microsoft
- 被誉为"所有演示之母"

### 1960s：NLS系统
**项目**：oN-Line System (NLS)
**功能**：
- 超文本文档系统
- 大纲编辑器
- 链接和引用
- 多人协同
- 视频会议原型

## 技术贡献总结

| 领域 | Engelbart的贡献 |
|------|----------------|
| 输入设备 | 鼠标发明 |
| 用户界面 | 图形界面、窗口概念 |
| 超文本 | 链接文档、超媒体 |
| 协同计算 | 视频会议、共享编辑 |
| 知识管理 | 大纲处理、结构化信息 |

## 增强人类智力（Augmentation）

### 核心理念
- 计算机应该增强人类智力，而非替代
- 工具应该放大人类能力
- 集体智慧的重要性

### "A Framework for Augmenting Human Intellect" (1962)
- 经典研究论文
- 阐述增强理论
- 影响后续HCI研究

## 对现代计算的影响

### Apple Macintosh
- Steve Jobs参观SRI，看到鼠标
- 直接影响Macintosh设计

### Xerox PARC
- PARC研究人员了解NLS
- 影响Alto系统

### Microsoft
- Windows系统借鉴概念
- Office软件中的超链接

### World Wide Web
- Tim Berners-Lee承认Engelbart影响
- 超文本概念的直接继承
